- asked ChatGPT for best way to pull from neuroglancer
   
   conclusion: recommends cloudVolume for downloading data because neuroglancer stores data
   in many precomputed chunks at multiple resolutions, cloudVolume package reads the info file, finds correct resolution,

   as a numpy array, all in one go. Apparently used by a many many people to view neuroglancer data.

   pip install cloud-volume
   if greenlet error pops up for this:
       pip install --only-binary :all: greenlet if 
   then try again

   for public data it is better to use https:// method instead of the gs:// method
   due to access credential errors
